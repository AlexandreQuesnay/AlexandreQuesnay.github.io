Projet événementiel : Après MMI ?

Cet événement avait pour objectif de reprendre contact avec les anciens étudiants du département d'elbeuf dans la formation MMI. Avec ce-ci, on pourrait mieux visualiser les possibilités d'avenir après cette formation.

Outils : E-mailling, Illustrator, Événementiel

Le projet

L'objectif principal de cet événement est de créer un événement qui regroupe les étudiants actuels et les anciens pour visualiser les parcours d'avenir après un BUT MMI.

Une fois cet objectif fixé, j'ai envoyé des mails aux anciens étudiants afin qu'ils puissent répondre à un questionnaire sur leur parcours. De plus, je devais les proposer de venir à l'IUT afin qu'ils participent à cet événement.

L'affiche

Pour faire la communication d'Après MMI, j'ai créé cette affiche qui regroupe la date, le nom et le but de l'événement afin d'attirer le plus personne.
