Dashboard pour la Haute Marne

Dans ce dashboard vous trouverez l'ensemble des données du Covid-19, sur les vaccinations, les nombres de personnes déclarer positive, etc. Ces données sont collectées du début du covid jusqu'à maintenant et elle permet de voir son avancement.

Objectifs

Pour faire simple, ce dashboard est la suite d'un devoir d'études pour voir l'évolution du Covid-19 dans le département du Haute Marne. On y retrouve tous type de graphique (histogramme, dorus, radar, etc) programmé avec charteJs et une base de données récupérer au sein du site du gouvernement.

Le deuxième objectif était plus centré sur le code source et les moyens que nous avons utiliser pour produire un dashboard cohérent, accessible et compréhensible pour un grand nombre de personnes.

Pour faire ce dashboard, j'ai utilisé le langages HTML, CSS, PHP, ChartJs et MySQLi
